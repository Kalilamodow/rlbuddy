# rlbuddy

An overlay which displays the ranks of everyone in your lobby. (and more coming soon!!!)

<img src="demo.jpeg">

> usage in casual (outdated)

## usage

First, make sure that the stats api is enabled. The instructions are [here](https://www.rocketleague.com/developer/stats-api#configuration). Set the packet rate to 1, but don't change the port. Then, start rocket league and the app.

You can manually open the overlay by holding `Alt`. It automatically pops up for a few seconds when the first countdown of a game starts.
